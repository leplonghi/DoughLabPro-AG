# Generated Artifacts Archive

This folder stores generated outputs and historical diagnostics that do not belong in the repository root.

Current buckets:
- `reports/`: audits, scans, and one-off generated summaries
- `style-research/`: intermediate JSON batches and extraction artifacts used during content migrations
- `translation-bundles/`: exported translation bundles and candidate key files
- `typescript-diagnostics/`: saved `tsc` outputs from previous repair passes

These files are preserved for traceability, but they are not part of the active runtime or primary build flow.
