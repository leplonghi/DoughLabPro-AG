# Archive Docs

This folder preserves historical documentation that no longer needs to live in the repository root.

Use it for:
- status snapshots
- migration reports
- implementation retrospectives
- generated documentation artifacts
- historical plans that were useful during execution but are no longer the primary source of truth

Active docs should remain in:
- [`../runbooks`](../runbooks)
- [`../architecture`](../architecture)
- other top-level docs under `docs/` that still guide current work
